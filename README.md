# Music_Player_WPF

Đồ án môn lập trình Windows - WPF

## Chức năng Hook bàn phím
```
  + Bài tiếp theo: Ctrl + Alt + N
  + Bài trước đó: Crtl + Alt + P
  + Tạm dừng / tiếp tục: Ctrl + Alt + D
  
```
